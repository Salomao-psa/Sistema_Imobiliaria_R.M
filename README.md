# Or-amento-de-Aluguel-R.M
Este projeto é uma aplicação desenvolvida em Python para automatizar o processo de orçamentos anuais da Imobiliária R.M. O sistema calcula valores de locação baseados em tipos de imóveis, adicionais de infraestrutura (quartos e vagas) e regras de descontos condicionais, gerando um cronograma financeiro detalhado.
